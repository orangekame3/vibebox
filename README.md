# vibebox

A sandbox repository for vibe coding with Claude Code.

## Setup

This repository is configured as a Claude Code sandbox with the following:

- `.claude/settings.local.json` - Permissive settings for sandbox experimentation
- `CLAUDE.md` - Project context for Claude Code
- `.gitignore` - Common patterns for multiple languages

## Usage

Start Claude Code in this directory and experiment freely:

```bash
claude
```

## Note

This is a sandbox environment. Feel free to experiment, prototype, and explore.
